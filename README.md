## Getting Started

### Requeriments
- Flutter v3.19.5
- Dart v3.3.3
- DevTools v2.31.1
- VS Code v1.88
- XCode (MacOS)
- Cocoapods (MacOS)
- Android SDK (Windows)

### Enviroments (.env.local, .env.develop, .env.production)
- API_URL=https://ip_computer
- DEBUG=true
- API_GOOGLE_MAPS=API_KEY

Es necesario crear los 3 archivos para manejar las variables de entorno, sino el proyecto no levanta
